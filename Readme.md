Для начала работы необходимо собрать образ из докер файла Dockerfile с помощью команды docker build --no-cache -t deepdoc_proj:1.0 . и запустим контейнер командой docker run -p 5000:5000 deepdoc_proj:1.0, после чего нужно перейти по ссылке http://127.0.0.1:5000, где вставить картинку приказ и нажать кнопку действия
